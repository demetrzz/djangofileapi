# DjangoFileAPI

## Функционал:

- загрузка, хранение, просмотр сохраненных файлов посредством API
- асинхронная обработка файлов с использованием celery
- проект "боевой" (готов к деплою)

## Стек технологий

- Django
- DjangoRestFramework
- Python
- PostgreSQL
- Redis
- Celery

## Стек деплоя:

- Docker
- Gunicorn
- Nginx
